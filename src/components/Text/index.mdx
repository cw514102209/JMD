# Text

## Basic usage

<Playground>
  <Text icon="android" theme="filled" title="title" description="description" />
</Playground>

## Properties

| 属性 | 说明 | 类型 | 默认值 |
| ---- | ---- | ---- | ---- |
| className | 自定义样式 | string，只支持 CSS Module | 无 |
| icon | 图标类型 | string | 无 |
| theme | 图标主题风格。可选实心、描线、双色等主题风格，适用于官方图标 | 'filled' | 'outlined' | 'twoTone' | 'outlined' |
| title | Text 的信息 | string | 无 |
| description | Text 的描述信息 | string | 无 |
