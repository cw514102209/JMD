## Properties

| 属性 | 说明 | 类型 | 默认值 |
| ---- | ---- | ---- | ---- |
| className | 自定义样式，只支持 CSS Module | string | 无 |
| type | 指定警告提示的样式，可选值为 'success', 'info', 'error', 'warning', 'default' | string | 'default' |
| showIcon | 是否显示辅助图标 | boolean | false |
| icon | 左侧图标 | string | 无 |
| message | 警告提示内容 | string | 无 |
| description | 警告提示的辅助性文字介绍 | string | 无 |
| closable | 默认不显示关闭按钮 | boolean | false |
| closeText | 自定义关闭按钮 | string 或 ReactNode | 无 |
